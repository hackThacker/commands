### Kali Linux Commands:

#### Basic Commands:

1. **ls**
   ```markdown
   List files and directories in the current folder.
   ```

2. **cd**
   ```markdown
   Change current directory.
   ```

3. **pwd**
   ```markdown
   Print the current working directory.
   ```

4. **mkdir**
   ```markdown
   Create a new directory.
   ```
